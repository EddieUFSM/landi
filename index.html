Ícones
Nosso projeto apresenta uma extensa coleção de ícones potencializada pelo FontAwesome. Com o FontAwesome, temos acesso a uma vasta biblioteca de ícones meticulosamente elaborados, abrangendo diversas categorias, desde ícones de interface do usuário até ícones de redes sociais, dispositivos, setas e muito mais.
Adicione seus próprios ícones é fácil! Basta adicionar seus arquivos SVG na pasta icons e, em seguida, executar o comando npm run generate:sprites-svg a partir da raiz do projeto.
Ao adicionar seus ícones personalizados, certifique-se de que os arquivos SVG contenham apenas o conteúdo do caminho () e os atributos xmlns e viewBox. Não inclua outras configurações no arquivo SVG, pois isso pode afetar o resultado final.
